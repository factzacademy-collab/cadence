   216ŌåÆSHARED DASHBOARD PRIMITIVES ŌĆö CONTRACT
   217ŌåÆFile: `src/components/dashboard/shared.tsx`
   218ŌåÆImport path: `@/components/dashboard/shared`
   219ŌåÆAll primitives are client-safe ("use client" at the top of the file).
   220ŌåÆ
   221ŌåÆ1. PageHeader({ title, description?, actions?, className? })
   222ŌåÆ   - Consistent view header. `title` is rendered inside an <h1>.
   223ŌåÆ   - `actions` is a flex row of buttons/links on the right (wraps on
   224ŌåÆ     mobile). Use this at the top of every view.
   225ŌåÆ
   226ŌåÆ2. StatCard({ label, value, delta?, deltaLabel?, icon?, accent?, spark? })
   227ŌåÆ   - KPI card with a label, big value, optional delta percentage
   228ŌåÆ     (positive = mint, negative = destructive), optional deltaLabel,
   229ŌåÆ     optional lucide `icon` rendered in a tinted swatch (use `accent`
   230ŌåÆ     like "text-primary", "text-coral", "text-mint", "text-plum"),
   231ŌåÆ     and optional `spark: number[]` rendered as a MiniSparkline.
   232ŌåÆ
   233ŌåÆ3. SectionCard({ title?, description?, actions?, children?, className?, bodyClassName? })
   234ŌåÆ   - Titled card wrapper used to group content inside a view. Header
   235ŌåÆ     is omitted if none of title/description/actions are passed.
   236ŌåÆ     `bodyClassName` defaults to `p-5`.
   237ŌåÆ
   238ŌåÆ4. StatusBadge({ status })
   239ŌåÆ   - Colored Badge for PostStatus union:
   240ŌåÆ     draft (muted), scheduled (primary), published (mint),
   241ŌåÆ     failed (destructive), in-review (amber).
   242ŌåÆ
   243ŌåÆ5. PostCard({ post })
   244ŌåÆ   - Compact post row: platform badges (stacked, +N overflow), 2-line
   245ŌåÆ     text snippet, relative or absolute time, StatusBadge, and a
   246ŌåÆ     kebab DropdownMenu with Edit (openComposer(post.id)), Duplicate
   247ŌåÆ     (useCreatePost), Delete (useDeletePost). All mutations fire
   248ŌåÆ     sonner toasts. Self-contained ŌĆö just drop it in any list.
   249ŌåÆ
   250ŌåÆ6. EmptyState({ icon?, title, description?, action?, className? })
   251ŌåÆ   - Tasteful dashed-border empty state with optional icon swatch,
   252ŌåÆ     title, description, and optional CTA node.
   253ŌåÆ
   254ŌåÆ7. SkeletonGrid({ count = 3, className? })
   255ŌåÆ   - Grid of shimmer cards (uses `.shimmer` utility from globals.css).
   256ŌåÆ
   257ŌåÆ8. Avatar({ name, color = "from-primary to-mint", size = "default", className? })
   258ŌåÆ   - Gradient initials avatar built on the shadcn <Avatar> primitive.
   259ŌåÆ     `color` is a tailwind gradient class string. `size` is
   260ŌåÆ     "sm" | "default" | "lg". Use this everywhere you need a user/team
   261ŌåÆ     avatar (NOT the platform badge).
   262ŌåÆ     NOTE: This is named `Avatar` and lives in
   263ŌåÆ     `@/components/dashboard/shared`. The shadcn Avatar lives in

   267ŌåÆ   - Tiny inline recharts LineChart (no axes). `data` is a plain

   269ŌåÆ
   270ŌåÆ10. MiniArea({ data, dataKey, color?, height?, formatX? })
   271ŌåÆ    - Small recharts AreaChart helper. `data` is an array of records,
